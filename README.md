# Kanban Board

### Used: 
  * ReactJS 16+ & React Router
  * Material UI
  * TypeScript
  * NodeJS (Express)
----------------------------------------

### Features: 
  * create/edit/delete card
  * create/edit/delete category
  * drag & drop cards between categories
  * save data with nodejs (rest api with express) into json file
